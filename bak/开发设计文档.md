# BlockCity 区块城市网站开发设计文档

## 一、项目概述

### 1.1 项目名称
BlockCity 区块城市 - 元宇宙先行者平台

### 1.2 项目定位
基于 Web3 和区块链思维的元宇宙城市平台，提供个性化的元宇宙生涯、去中心化组织和社交体验。

### 1.3 技术栈
- **前端框架**: Vue.js (基于 uni-app 框架)
- **UI组件**: 自定义组件 + uni-app 原生组件
- **样式**: CSS3 + 响应式设计
- **构建工具**: Webpack
- **分析工具**: 百度统计

## 六、业务逻辑详细设计

### 6.1 核心业务模型

#### 6.1.1 平台定位
BlockCity区块城市是一个基于区块链技术的去中心化元宇宙城市平台，核心特点：
- **去中心化自治组织(DAO)**: 用户参与城市治理决策
- **数字身份(DID)**: 每个用户拥有唯一的去中心化数字身份
- **虚拟地产**: 用户可购买、交易、开发虚拟地块
- **虚实结合**: 虚拟资产与现实价值互通

#### 6.1.2 平台资质
- 国家网信办区块链信息服务备案
- 首批《元宇宙参考架构》国家标准应用试点单位
- 运营主体：区块城市（北京）科技有限公司 / 广州兆谷信息产业集团有限公司

### 6.2 DAO治理机制

#### 6.2.1 治理架构
```
DAO治理体系
├── 立法权
│   ├── 用户提案发起
│   ├── 规则修改投票
│   └── 核心参数调整
├── 执行权
│   ├── DAO治理委员会
│   ├── 技术委员会审核
│   └── 社区仲裁庭复核
└── 监督权
    ├── 资金流向透明
    ├── 智能合约自动执行
    └── 链上存证可查
```

#### 6.2.2 治理流程
```
提案发起 → 社区讨论 → 投票表决 → 技术审核 → 仲裁复核 → 执行落地
    ↓          ↓          ↓          ↓          ↓          ↓
  30%用户    公开辩论    51%通过    代码审查    合规检查    智能合约
  提案支持              代币投票    漏洞检测    72h公示    自动执行
```

#### 6.2.3 治理范围
| 治理领域 | 具体内容 | 投票门槛 |
|---------|---------|---------|
| 城市规划 | 地标建筑、功能区域划分 | 30%提案+51%投票 |
| 规则修改 | 交易手续费率(上限5%)、基金分配比例 | 51%投票通过 |
| 项目众投 | 基建、文旅、教育等实体项目 | 按贡献值分配收益 |
| 资金调用 | 城市基金使用申请 | 三重验证机制 |

#### 6.3.2 地块分类

| 分类 | 描述 | 价格范围 | 特点 |
|-----|------|---------|------|
| 核心区域 | DAO治理中心、虚拟商业圈 | 较高 | 增值空间大 |
| 潜力发展区 | 官方规划的发展区域 | 中等 | 成长性好 |
| 新人专属区 | 新用户入门区域 | 99元起 | 附赠成长礼包 |
| 限量地块 | 带独特景观或功能权限 | 较高 | 收藏价值 |

#### 6.3.3 交易流程
```
注册账号 → 选择城市 → 浏览地块 → 确认购买 → 支付款项 → 确权上链
    ↓         ↓         ↓         ↓         ↓         ↓
 生成DID   337个城市  虚拟沙盘   价格确认   数字人民币  区块链存证
          可选择     预览规划   风险提示   支付接口   永久有效
```

#### 6.3.4 地块运营
- **每日签到**: 连续7天签到解锁人气值加速包
- **搭建标识**: 在地块搭建基础标识积累人气值
- **DAO投票**: 参与城市治理获得贡献值
- **出租商铺**: 将地块出租给品牌方获取租金
- **举办活动**: 在地块举办虚拟活动获取收益

### 12.1 XSS防护

```javascript
// 输入过滤
function escapeHtml(text) {
    return text
        .replace(/&/g, '&amp;')
        .replace(/</g, '&lt;')
        .replace(/>/g, '&gt;')
        .replace(/"/g, '&quot;')
        .replace(/'/g, '&#039;');
}
```
